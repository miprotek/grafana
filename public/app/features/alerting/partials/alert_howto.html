Alarm hinzufügen

Warnungen werden auf der Registerkarte Alarm Tab / Meldung Tab in jedem Dashboard-Diagrammfenster hinzugefügt und konfiguriert, sodass Sie mithilfe vorhandener Abfragen eine Warnung erstellen und visualisieren können.

Um Änderungen an den Warnungen zu vermeiden, sollten Sie das Dashboard speichern.
